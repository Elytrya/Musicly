# Musicly
Musiqly is a client-server Minecraft mod that lets you play music from Spotify, YouTube, SoundCloud, Yandex Music, and more directly in-game. Supports streams, TTS, and multiple audio formats without leaving the game.

**Project is currently in development and is expected to be available around mid-April.**
**Проект находится в разработке и планируется к выпуску примерно к середине апреля.**
